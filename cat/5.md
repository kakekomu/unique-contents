### 〈パワハラの種類と定義を知って、労働トラブルを防ごう〉
企業における円滑な組織運営のためには、上司と部下の上下関係は重要です。しかし、優位な立場にある人間は、業務上の上下関係を超えた高圧的な態度をとってしまう場合が少なくありません。

労働トラブルになりやすいパワハラの定義と、パワハラが起きない職場づくりのために有効な対策を紹介します。

#### パワハラとは？指導との違いは？
パワハラは、被害に遭った人が精神を病みかねない許しがたい行為です。しかし、仕事をするうえでは、上司から部下にある程度の指示や指導をしなくてはなりません。パワハラと指導の違いはどのような点にあるのでしょうか？

ここでは、パワハラの定義と指導との違いについて解説します。

- **パワハラとは**

パワハラの定義は3つ、累計は6つです。以下、説明させていただきます。

厚生労働省によると、職場における以下の3つの条件を満たす行為がパワハラと定義されています。

**①優越的な関係を背景とした行為であること**

**②業務上必要かつ相当な範囲を超えたものであること**

**③労働者の就業環境が害されるものであること**

|  No.  |  説明  |
| ---- | ---- |
|  ①  | 「優越的な関係」とは、職務上の地位が上位の者から行われることを意味します。これは、上司から部下による行為だけを指すのではありません。同僚や部下であっても、その者が持つ知識や技術がなければ仕事が立ち行かなかったり、集団の力を利用することで抵抗・拒否することが困難な関係である場合は、「優越的な関係」とみなされます。 |
|  ②  | 「業務上必要かつ相当な範囲を超えたもの」とは、社会通念に照らし、その行為が業務上ふさわしくないレベルであることを意味します。「必要かつ相当な範囲を超えて」いるかどうかは、仕事内容やその行為が発生した状況などを考慮して総合的に判断しなければなりません。 |
|  ③  | 「労働者の就業環境が害される」とは、その行為により労働者が心身ともに苦痛を与えられ、就業環境が不快なものとなり、仕事にも支障が出てしまう状況を指します。人によって苦痛の感じ方はさまざまなので、「一般的な労働者が不快に感じる程度」を基準に判断されます。 |

- **パワハラの６つの類型**

パワハラは大きく分けて以下の6つに分類されます。
|  類型  |  説明  |
| ---- | ---- |
|  ①身体的な攻撃  | 殴る、蹴る、叩くなど、直接的に身体を攻撃する行為を指します。 |
|  ②精神的な攻撃  | 大勢の前で叱責したり、長時間に渡り怒鳴り続けるなどの行為を指します。 |
|  ③人間関係から孤立させる  | 一人だけ別室に席を移される、コミュニケーションに参加させないなどの行為を指します。 |
|  ④過大な要求  | 明らかに達成が不可能な業務量を与え、精神的に追い詰める行為を指します。 |
|  ⑤過小な要求  | 誰でもできるような仕事のみ与え、自信を失わせたり、能力を発揮する機会を奪う行為を指します。 |
|  ⑤個人を否定する  | 個人の属性や家族などを否定する行為を指します。 |

- **パワハラはなぜ指導といえないのか？**

このように、パワハラには明確な定義があり、行為の悪質性や被害者が受けるダメージの大きさなどを考えれば、指導の範囲を超えていることがわかります。

ただし、指導であっても、状況や言い方によって相手は自身を否定されているように感じたり、疎外されているように感じたりする場合があります。特に上述したパワハラの6つの類型にあてはまる行為は誤解を生みやすいので注意しましょう。

パワハラと指導の違いは、「相手のことを想っているかどうか」にあります。適切な指導をするためには、「（指導の相手が）今よりも仕事がしやすくなるため」「今よりも成長するため」といったポジティブな目的や相手のメリットを見据えたうえで行うことが大切です。また、すでに起きてしまったことを批判するのではなく、より良い未来に向けて改善していく姿勢で話しましょう。

一方で、いわゆる「愛のムチ」といわれるような、本心は相手を応援していてもわざと乱暴な𠮟り方をする行為を指導と思っている上司もいるようです。このような指導は、部下に本心が伝わらない限りはパワハラとみなされる可能性が高いため注意しましょう。

#### パワハラを放置するとどんな悪影響があるか？
パワハラを放置することが結果として会社のためにならないことを簡単に説明してください。

パワハラは個人を追い詰めるだけでなく、組織全体に悪影響を及ぼします。職場内でパワハラが発生しているにもかかわらず放置し続けた場合、企業にはどのようなデメリットがあるのでしょうか？

- **社員のストレスが溜まる？**

社員のストレスが溜まると、パワハラの悪化や精神疾患につながることを説明してください。

パワハラ被害者は過大なストレスを抱えています。被害者が我慢し続けても状況が改善する可能性は低く、企業側がなんらかの対策をしないかぎりパワハラはエスカレートしていくでしょう。結果的に被害者が精神疾患にかかり、健康な生活ができなくなってしまうかもしれません。

また、被害者本人だけでなく、周囲の社員にとっても、パワハラが見て見ぬふりされる職場は不安感につながります。

- **組織の生産性が悪化する**

組織の生産性が悪化すること、それに対して叱責しても何も変わらないことを示してください。

パワハラの被害によって心身にダメージを受けている状態では、仕事に集中できなくなりミスが増えます。企業や自分の成長のために頑張ろうという気持ちも生まれにくいでしょう。そのため、パワハラを放置している職場は、生産性が低下することが少なくありません。このような場合、生産性が下がったことに対して、さらに叱責を受ける回数が増えるなど、パワハラの悪循環に陥ってしまうことも考えられます。

- **企業全体のイメージダウンにつながる**

パワハラが公になると会社全体のイメージダウンにつながるが、パワハラを隠すとそれ以上の悪影響になることを説明してください。

パワハラは組織内部の問題ですが、組織の不穏な雰囲気は外部からもわかるものです。特に、休職者や退職者の増加は数値データとして外部にも伝わるため、パワハラが蔓延している職場環境を隠し通すのは不可能といって良いでしょう。

また、ひどいパワハラで社員が心身を病んだ場合、労働災害として認定されることもあります。近年ではパワハラが訴訟に発展するケースも多く、そうなれば企業のイメージダウンは避けられません。

#### パワハラではない指導を行うために
部下を指導するにあたっては、パワハラとみなされることは避けなければなりません。パワハラではない指導を行うためにはどのようなポイントに気を付ければ良いのでしょうか。

- **なぜ指導をするのか部下にきちんと伝える**

指導は、部下が失敗したりミスしたりした時に行うことが多いです。しかし、失敗を叱責するだけでは部下は自身を否定されたと思ってしまうでしょう。なぜこの失敗が問題なのか、どのように改善すると良いのかなど、指導の背景や理由をしっかりと伝えることが大切です。

- **ポジティブな言葉を用いる**

指導は、部下の将来を考え、成長を促す目的で行うものです。そのため、できるだけポジティブな言葉を選び、穏やかで明るい雰囲気になるよう心掛けましょう。人間は自身を認めてくれる人の意見ほど耳に入れるものです。相手を肯定し、前向きな言葉をかけるようにしましょう。

- **相手の意見も聞く**

いくら正当な指導であっても、頭ごなしに「こうしなさい」というだけでは威圧感を与え、相手は強制されたと感じるかもしれません。指導するにあたっては、まずは部下がどのような考えで業務を行っているのか意見を聞きましょう。その意見を否定せず、「このやり方の方が良いよ」といったアドバイスをすれば、納得してもらいやすいです。

#### パワハラへの対策として企業ができること
人がパワハラをしてしまうのは、個人の性格や性質が大きな要因です。しかし、職場環境の隠れた問題が関係している場合も少なくありません。日頃から社員の働き方に気を配ることで、パワハラの発生を防止することが可能です。ここでは、パワハラ対策として企業ができることを解説します。

- **訴えられたら誠実に対応する**

訴えられても慌てず法に従い対応すべきことを説明してください。ここで弁護士を立てる有用性も解説します。

近年、パワハラ被害者によって企業を相手に訴訟が起こされるケースが増えています。2020年6月に施行された「労働施策総合推進法（通称：パワハラ防止法）」では、企業に対してパワハラ防止方針の明確化や相談窓口の整備、パワハラに関する労使紛争への対応義務などについて定めています。つまり、職場でパワハラ被害を受けた場合、企業の義務違反を追求できるようになったのです。パワハラ防止法に罰則はありませんが、厚生労働省から指導や勧告を受ける可能性があり、企業のイメージダウンや信頼の失墜につながりかねません。

そのため、紛争や訴訟に発展した場合は、法律に従い誠実に対応することが大切です。自社だけで対応しようとせず、弁護士を立て自社の対応方針についてアドバイスをもらうべきでしょう。

- **労働環境・人材配置の改善**

パワハラは労働環境や人材配置を改善することである程度防止することができます。例えば、重いノルマや長時間労働を課している職場では、社員の心身にストレスがかかり、人間関係も荒みがちです。社員一人一人の能力に合わせ適正な労働量を管理することで、ワークライフバランスを向上させましょう。また、パワハラに発展しそうな人間関係に注意し、必要に応じて人材配置を変更するなど、組織運営に気を配る必要があります。

このように職場を適切にマネジメントするためには労働基準法、労働契約法などの労働法の理解が必要不可欠です。自社の状況が法律に違反していないか、弁護士など法律の専門家に確認してもらうと良いでしょう。

- **研修などによる周知**

人によっては、パワハラに該当する行為を、パワハラと知らずに行ってしまう場合があります。本人はちょっとした冗談のつもりであったり、コミュニケーションの一環と思っていたりするため、こういったパワハラはなかなか改善されません。そのため、社内で研修の機会を設け、どのような行為がパワハラにあたるのかをしっかり周知しましょう。研修ではなるべく具体例を提示し、わかりやすい内容にすることが大切です。

- **ストレスチェックの実施**

労働者が常時50名以上いる職場では、ストレスチェックの実施義務があります。社員の状況を把握するために、実施義務のない企業でもストレスチェックを行うことが望ましいです。また、定期的に全社に向けてサーベイ（従業員の意識調査）を実施するなど、小さな芽のうちにパワハラに対応できるようにしましょう。

- **相談窓口の設置**

パワハラ防止法では、企業に対して、パワハラに関する相談窓口の設置を義務付けています。ハラスメント相談窓口は、社員の相談しやすさを第一に考え、プライバシーや秘密保持に配慮することが大切です。セクハラやマタハラなど、その他の相談についても総括的に対応できる窓口にすると良いでしょう。

また、相談内容によって、相談者や関係者が不利益な取り扱いを受けることがないよう、中立的な立場を確立しなければなりません。社員に対して相談窓口の存在を周知することも大切です。

- **パワハラ対策で弁護士の協力を得るべき理由**

パワハラは、被害者の心身を害し、健全な労働環境を損なわせる卑劣な行為です。一連のパワハラ被害のなかにはさまざまな問題が隠れており、蓋を開けてみれば長時間労働や業務過多などの問題があるなど、企業側が責任を追及されるケースも少なくありません。

そのため、パワハラ防止のための対策から、パワハラが起こってしまったあとの対応まで、自社だけで対応しようとせず、法律の専門家である弁護士としっかりと連携することが重要です。日頃から労働環境全体の整備を心掛け、いつでも相談できるコンサルティング能力に長けた弁護士とつながりを持っておくと安心でしょう。

#### パワハラ対策をするメリット
パワハラ対策をすべきは、社員との法律トラブル回避だけにとどまりません。パワハラの少ない会社でも更なるメリットがもたらされます。

- **労働生産性が上がる**

労働生産性と人件費の関係について簡単に説明お願いいたします。

労働生産性は社員のパフォーマンス発揮の度合いと密接に関わります。パワハラが横行している職場では個人が抑圧されるだけでなく組織の機能不全も見られます。一方でパワハラが起きない職場環境とは、社員が気持ちよくコミュニケーションを取れる場を意味します。

労働生産性を高めたいなら、まずは社員を邪魔する要因の解決から始めましょう。余計な残業や経費使用の削減も期待されます。

- **良い人が集まってくる**

離職率の低下と、採用できるレベルが上がることについて説明お願いします。

職場環境は社員にとって大きな評価基準で、人間関係が転職・退職理由の多くを占めることはよく知られています。逆に言えば、人間関係での問題が少ないというだけで社員の離職率低下に繋がります。

採用活動においてもパワハラの有無は求職者にとって無視できないポイントです。良い人材が入社後も働き続けてくれる会社であることは採用コストを大幅に削減してくれるでしょう。

#### まとめ

上司からの指導や叱責は、相手にとってメリットがあるからこそ認められるもの。それが行き過ぎれば、単なる嫌がらせになってしまいます。パワハラが蔓延る職場は社員の生産性が下がり、採用活動も難航するでしょう。

現代の法律に合った職場環境を作り、その上で社員の成果が上がる状態を実現したいなら、ぜひコンサルティング能力に優れた弁護士の助けを得ましょう。

### 〈労働審判手続き〉 
#### １．労働審判手続きの概要
・労働審判手続は，解雇や給料の不払など，個々の労働者と事業主との間の労働関係のトラブルを，その実情に即し，迅速，適正かつ実効的に解決するための手続です。

・訴訟手続とは異なり非公開の手続です。
#### ２．労働審判手続きの特徴
**（１）労働関係の専門家による関与**

 
　労働審判手続は，労働審判官（裁判官）１名と労働審判員２名で組織する労働審判委員会が行います。


　労働審判員は，雇用関係の実情や労使慣行等に関する詳しい知識と豊富な経験を持つ者の中から任命され，中立かつ公正な立場で，審理・判断に加わります。

 **（２）迅速な手続**

　原則として3回以内の期日で審理を終えることになっているため，迅速な解決が期待できます。

　平成18年から令和4年までに終了した事件について，平均審理期間は81.2日であり，66.9％の事件が申立てから3か月以内に終了しています。

 **（３）事案の実情に即した柔軟な解決**

  　労働審判委員会は，まず調停という話合いによる解決を試み，話合いがまとまらない場合には，審理の結果認められた当事者間の権利関係と手続の経過を踏まえ，事案の実情に即した判断（労働審判）を行い，柔軟な解決を図ります。

**（４） 異議申立てによる訴訟移行**

　労働審判に不服のある当事者は，異議申立てをすることができます。適法な異議申立てがなされた場合は，労働審判は効力を失い，訴訟手続に移行します。

#### ３．労働審判手続きの流れ
![02-tetuduki](https://github.com/kakekomu/unique-contents/assets/116237570/eab70a6d-facc-488d-9fe6-34c4ac7c9090)
#### ４．労働審判手続きの利用にあたっての留意点
・トラブルの内容が複雑で，限られた期日の中で審理を終えることが難しそうな事案にはなじみません。トラブルの解決に労働審判手続が適していないと認められるときは，労働審判委員会が事件を終了させることがあり，この場合は，訴訟手続に移行します。

・3回以内の期日で集中して審理を行うためには，当事者は，早期に的確な主張・立証を行うことが重要です。

・申立書には，当事者間の交渉など申立てに至る経緯の概要も記載する必要があるため，労働審判手続の申立て前に当事者間で交渉を行ったり，行政機関等によるあっせん手続を行ったりしておくことが求められます。

・労働関係のトラブルの解決方法には，労働審判手続以外にも様々な手続があります。それぞれの手続の特徴と事案の実情等を踏まえて，どの手続を利用するのが良いのかを十分に検討した上で手続を選択してください。
#### ５．弁護士への相談について
　労働審判手続は，原則として3回以内の期日で審理を終結することになるため，申立ての段階から十分な準備をして，充実した内容の申立書と必要な証拠を提出することが重要です。

　また，当事者双方は，期日において口頭で言い分を述べることが原則とされていますから，申立人は，相手方から提出される答弁書や証拠をしっかりと検討し，期日において的確な主張（言い分）を述べ，証拠を提出することが重要です。

　さらに，トラブルの内容が労働審判手続による解決に適したものかどうかを見極めることも重要です。

　弁護士に依頼するかどうかは，最終的には，自分の意思で決めていただくことになりますが，このように，労働審判手続による解決に適した事案かどうかを適切に見極め，申立ての段階から十分な準備をし，期日において状況に応じた的確な主張，立証を行うためには，必要に応じて，法律の専門家である弁護士に依頼することが望ましいでしょう。
